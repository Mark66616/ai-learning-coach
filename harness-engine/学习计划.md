**第二段：fixture（30 min）**

fixture 是 pytest 的核心机制——在测试前准备数据，测试后清理。这与 harness 的"测试数据准备"概念直接对应。

`test_fixtures.py`：

```python
import pytest

# 基本 fixture：提供测试数据
@pytest.fixture
def sample_emails():
    return [
        {"text": "会议改到下午3点", "label": "work"},
        {"text": "你的快递已签收", "label": "personal"},
        {"text": "限时优惠，点击领取", "label": "spam"},
        {"text": "项目报告请本周五前提交", "label": "work"},
    ]

# 使用 fixture：作为函数参数传入
def test_email_classification(sample_emails):
    """这个测试和 harness 的测试用例管理直接对应"""
    for email in sample_emails:
        result = classify_email(email["text"])
        assert result is not None
        assert result in ["work", "personal", "spam"]

# fixture 的 scope：控制创建频率
@pytest.fixture(scope="module")  # 整个模块只创建一次
def model_connection():
    """模拟模型连接，整个模块共享"""
    print("\n[Setup] 建立模型连接...")
    conn = {"model": "gpt-4", "connected": True}
    yield conn  # yield 之前是 setup，之后是 teardown
    print("\n[Teardown] 关闭模型连接...")
    conn["connected"] = False

def test_model_connected(model_connection):
    assert model_connection["connected"] is True

def test_model_name(model_connection):
    assert model_connection["model"] == "gpt-4"
```

`conftest.py`（共享 fixture）：

```python

**第三段：参数化 + 命令行（30 min）**

参数化是 harness 批量测试的核心——一条测试逻辑，跑多组数据。

`test_parametrize.py`：

```python
import pytest

# 参数化：用一组数据跑同一个测试
# 这就是 harness "批量测试"的 pytest 版本
@pytest.mark.parametrize("text,expected", [
    ("会议改到下午3点", "work"),
    ("你的快递已签收", "personal"),
    ("限时优惠，点击领取", "spam"),
    ("", ""),  # 边界：空输入
    ("asdfghjkl", "unknown"),  # 边界：无意义输入
])
def test_classify_single(text, expected):
    result = classify_email(text)
    assert result == expected

# 参数化 + ID：给每组数据起名字，方便定位失败
@pytest.mark.parametrize("text,expected", [
    ("会议改到下午3点", "work"),
    ("限时优惠", "spam"),
], ids=["work_email", "spam_email"])
def test_classify_with_id(text, expected):
    result = classify_email(text)
    assert result == expected

# 多参数组合
@pytest.mark.parametrize("model", ["gpt-4", "gpt-3.5", "claude-3"])
@pytest.mark.parametrize("temperature", [0.0, 0.7, 1.0])
def test_model_config(model, temperature):
    """测试不同模型 × 不同温度的组合"""
    config = {"model": model, "temperature": temperature}
    assert config["model"] == model
    assert 0 <= config["temperature"] <= 2
```

#### 详细学习内容

**1. 概念溯源（20 min）**

Harness 这个词来自马具——"套在马身上的装备"，引申到软件工程就是"套在被测系统外面的脚手架"。

演进路径：

```
传统软件 test harness          AI model harness
┌──────────────┐              ┌──────────────┐
│ 测试输入     │              │ Prompt 模板  │
│     ↓        │              │     ↓        │
│ 被测程序     │     →        │ LLM 模型     │
│     ↓        │              │     ↓        │
│ 输出比对     │              │ 输出解析     │
│     ↓        │              │     ↓        │
│ 通过/失败    │              │ 多维评估     │
└──────────────┘              └──────────────┘
```

关键区别：传统 harness 是二元判断（通过/失败），AI harness 需要多维度评估（准确率、相关性、安全性等），因为 LLM 输出没有唯一正确答案。

**2. 核心架构四层模型（30 min）**

```
┌─────────────────────────────────────────────┐
│              评估层 (Evaluation)              │
│  指标计算 · 结果对比 · 报告生成 · 退化检测    │
├─────────────────────────────────────────────┤
│              输出层 (Output)                  │
│  响应解析 · 结构化提取 · 错误捕获 · 重试      │
├─────────────────────────────────────────────┤
│              模型层 (Model)                   │
│  API 封装 · 本地模型 · 采样参数 · 限速控制    │
├─────────────────────────────────────────────┤
│              输入层 (Input)                   │
│  Prompt 模板 · 数据加载 · 变量注入 · Few-shot │
└─────────────────────────────────────────────┘
```

每层详解：

**输入层** — 解决"怎么喂数据给模型"

- Prompt 模板：把可变部分（用户输入）和固定部分（系统指令）分离
- 数据加载：从文件/数据库/API 读取测试用例
- 变量注入：把测试用例的字段填入模板
- Few-shot 组织：管理示例的选择和拼接

```python
# 输入层示意
class InputLayer:
    def __init__(self, template_path: str):
        self.template = load_template(template_path)
        self.test_cases = []

    def load_dataset(self, path: str):
        """加载测试数据集（JSONL 格式）"""
        with open(path) as f:
            self.test_cases = [json.loads(line) for line in f]

    def render_prompt(self, test_case: dict) -> str:
        """把测试用例填入模板"""
        return self.template.render(**test_case)
```

**模型层** — 解决"怎么调用模型"

- API 封装：统一接口，屏蔽不同模型 API 差异
- 采样参数管理：temperature、top_p、max_tokens 等
- 限速控制：防止 API 调用超限
- 本地模型支持：通过 vLLM / Ollama 跑本地模型

```python
# 模型层示意
class ModelLayer:
    def __init__(self, model_name: str, **sampling_params):
        self.model = model_name
        self.params = sampling_params  # temperature, top_p, etc.
        self.rate_limiter = RateLimiter(max_calls=60)  # 每分钟60次

    async def generate(self, prompt: str) -> str:
        async with self.rate_limiter:
            response = await client.chat.completions.create(
                model=self.model,
                messages=[{"role": "user", "content": prompt}],
                **self.params
            )
            return response.choices[0].message.content
```

**输出层** — 解决"怎么处理模型返回"

- 响应解析：从原始文本中提取结构化信息
- 错误捕获：API 错误、格式错误、超时

```python
# 输出层示意
class OutputLayer:
    def parse(self, raw_response: str, expected_format: str) -> dict:
        """解析模型输出为结构化数据"""
        if expected_format == "json":
            return self._parse_json(raw_response)
        elif expected_format == "classification":
            return {"label": raw_response.strip().lower()}
        else:
            return {"text": raw_response}

    def _parse_json(self, text: str) -> dict:
        try:
            # 尝试从 markdown code block 中提取
            if "```json" in text:
                json_str = text.split("```json")[1].split("```")[0]
            else:
                json_str = text
            return json.loads(json_str)
        except (json.JSONDecodeError, IndexError) as e:
            return {"_parse_error": str(e), "_raw": text}
```

**评估层** — 解决"怎么判断输出好不好"

- 指标计算：根据任务类型选择合适的指标
- 结果对比：与金标准对比、与基线对比
- 报告生成：生成可读的评估报告
- 退化检测：检测性能是否下降

```python
# 评估层示意
class EvaluationLayer:
    def evaluate(self, predictions: list, references: list) -> dict:
        results = {}
        # 准确性
        results["exact_match"] = self._exact_match(predictions, references)
        # 语义相似度
        results["semantic_similarity"] = self._semantic_sim(predictions, references)
        # 效率
        results["avg_latency_ms"] = self._avg_latency()
        # 生成报告
        return results

    def _exact_match(self, preds, refs):
        correct = sum(1 for p, r in zip(preds, refs) if p == r)
        return correct / len(preds) if preds else 0.0
```

class PromptManager:
    def __init__(self, config_path: str):
        self.config = load_yaml(config_path)
        self.templates = {}

    def get_template(self, task_name: str, version: str = None) -> Template:
        if version is None:
            version = self.config[task_name]["current_version"]
        template_file = self.config[task_name]["versions"][version]["template"]
        return load_template(f"{task_name}/{template_file}")

    def get_ab_variant(self, task_name: str) -> tuple[str, Template]:
        """A/B 测试：随机返回一个变体"""
        ab_config = self.config[task_name].get("ab_test", {})
        if not ab_config.get("active"):
            version = self.config[task_name]["current_version"]
            return version, self.get_template(task_name, version)

        import random
        variants = ab_config["variants"]
        weights = [ab_config["traffic_split"][v] for v in variants]
        chosen = random.choices(variants, weights=weights)[0]
        return chosen, self.get_template(task_name, chosen)
```

```python
# Exact Match：精确匹配，最严格
def exact_match(pred: str, ref: str) -> float:
    return 1.0 if pred.strip().lower() == ref.strip().lower() else 0.0

# F1 Score：部分匹配，适合多标签分类
from sklearn.metrics import f1_score
def f1(pred_labels: list, ref_labels: list) -> float:
    return f1_score(ref_labels, pred_labels, average="weighted")

# BLEU：机器翻译常用，衡量 n-gram 重叠
from sacrebleu import corpus_bleu
def bleu(preds: list, refs: list) -> float:

from rouge_score import rouge_scorer
def rouge(pred: str, ref: str) -> dict:
    scorer = rouge_scorer.RougeScorer(["rouge1", "rougeL"], use_stemmer=True)
    return scorer.score(ref, pred)

```python
# Embedding 相似度：用向量距离衡量语义接近程度
import numpy as np
from openai import OpenAI
client = OpenAI()

def semantic_similarity(pred: str, ref: str) -> float:
    """计算预测和参考答案的余弦相似度"""
    resp = client.embeddings.create(
        model="text-embedding-3-small",
        input=[pred, ref]
    )
    vec_pred = np.array(resp.data[0].embedding)
    vec_ref = np.array(resp.data[1].embedding)
    return float(np.dot(vec_pred, vec_ref) /
                 (np.linalg.norm(vec_pred) * np.linalg.norm(vec_ref)))

```python
import time

class EfficiencyTracker:
    def __init__(self):
        self.records = []

    def record(self, call_id: str, latency_ms: float,
               input_tokens: int, output_tokens: int, cost: float):
        self.records.append({
            "call_id": call_id,
            "latency_ms": latency_ms,
            "input_tokens": input_tokens,
            "output_tokens": output_tokens,
            "cost": cost
        })

    def summary(self) -> dict:
        latencies = [r["latency_ms"] for r in self.records]
        return {
            "count": len(self.records),
            "avg_latency_ms": sum(latencies) / len(latencies),
            "p50_latency_ms": percentile(latencies, 50),
            "p95_latency_ms": percentile(latencies, 95),
            "p99_latency_ms": percentile(latencies, 99),
            "total_tokens": sum(r["input_tokens"] + r["output_tokens"] for r in self.records),
            "total_cost": sum(r["cost"] for r in self.records),
        }
```

**安全指标**（输出安不安全）：

- 毒性检测：用 Perspective API 或专门模型
- 幻觉率：输出中的事实性陈述是否可验证
- 越狱成功率：对抗用例是否绕过了安全限制

**3. LLM-as-Judge 完整实现（25 min）**

```python
import json
from openai import OpenAI

class LLMJudge:
    def __init__(self, judge_model: str = "gpt-4o", temperature: float = 0.0):
        self.client = OpenAI()
        self.judge_model = judge_model
        self.temperature = temperature

    def evaluate(self, input_text: str, reference: str,
                 prediction: str, rubric: dict) -> dict:
        prompt = self._build_prompt(input_text, reference, prediction, rubric)
        response = self.client.chat.completions.create(
            model=self.judge_model,
            temperature=self.temperature,
            messages=[{"role": "user", "content": prompt}],
            response_format={"type": "json_object"}
        )
        return json.loads(response.choices[0].message.content)

    def _build_prompt(self, input_text, reference, prediction, rubric):
        rubric_str = yaml.dump(rubric, allow_unicode=True)
        return f"""你是一个专业的评估员。请根据以下标准评估模型的输出。

## 评估标准
{rubric_str}

## 测试输入
{input_text}

## 参考答案
{reference}

## 模型输出
{prediction}

## 请评估
请对每个维度打分（0-5），并简要说明理由。

输出 JSON 格式：
{{
  "scores": {{
    "准确性": <0-5>,
    "置信度合理性": <0-5>
  }},
  "reasoning": "<简要说明>"
}}
"""

    def batch_evaluate(self, cases: list, rubric: dict) -> list:
        """批量评估"""
        results = []
        for case in cases:
            result = self.evaluate(
                case["input"], case["expected"],
                case["prediction"], rubric
            )
            result["case_id"] = case["id"]
            results.append(result)
        return results
```

from jinja2 import Environment, FileSystemLoader

class InputLayer:
    def __init__(self, templates_dir: str = "templates"):

    def load_dataset(self, path: str) -> list[dict]:
        """加载 JSONL 格式的测试数据集"""

        with open(path, "r", encoding="utf-8") as f:
            for line in f:
                line = line.strip()
                if line:
                    cases.append(json.loads(line))
        return cases

    def render_prompt(self, template_name: str, variables: dict) -> str:
        """渲染 prompt 模板"""

        return template.render(**variables)

`harness/model_layer.py`：

```python
import time
from openai import OpenAI

class ModelLayer:
    def __init__(self, model: str = "gpt-4o-mini", temperature: float = 0.0):
        self.client = OpenAI()
        self.model = model
        self.temperature = temperature

    def generate(self, prompt: str) -> dict:
        """调用模型，返回结果和元数据"""
        start = time.time()
        response = self.client.chat.completions.create(
            model=self.model,
            temperature=self.temperature,
            messages=[{"role": "user", "content": prompt}]
        )
        latency_ms = (time.time() - start) * 1000

import re
import json

class OutputLayer:
    def parse(self, raw_response: str, task_type: str = "classification") -> dict:
        """解析模型输出"""
        if task_type == "classification":
            return self._parse_classification(raw_response)
        elif task_type == "json":
            return self._parse_json(raw_response)
        else:
            return {"text": raw_response.strip()}

    def _parse_classification(self, text: str) -> dict:
        """从文本中提取分类标签"""
        text = text.strip().lower()
        # 尝试直接匹配
        labels = ["work", "personal", "spam", "unknown", "refused"]
        for label in labels:
            if label in text:
                return {"label": label, "raw": text}
        # 尝试从 JSON 中提取
        try:
            data = json.loads(text)
            if "label" in data:
                return {"label": data["label"], "raw": text}
        except json.JSONDecodeError:
            pass
        return {"label": "parse_error", "raw": text}

    def _parse_json(self, text: str) -> dict:
        try:
            # 尝试从 code block 中提取
            match = re.search(r"```(?:json)?\s*(.*?)```", text, re.DOTALL)
            json_str = match.group(1) if match else text
            return json.loads(json_str)
        except (json.JSONDecodeError, AttributeError) as e:
            return {"_parse_error": str(e), "_raw": text}

```python
from collections import defaultdict

class Evaluator:
    def __init__(self):
        self.results = []

    def evaluate_case(self, case: dict, prediction: dict, metadata: dict) -> dict:
        """评估单个用例"""
        expected = case.get("expected", "")
        pred_label = prediction.get("label", "")

        # Exact Match
        exact_match = 1.0 if pred_label == expected else 0.0

        return {
            "case_id": case.get("id", ""),
            "category": case.get("category", ""),
            "subtype": case.get("subtype", ""),
            "input": case.get("text", "")[:100],  # 截断显示
            "expected": expected,
            "predicted": pred_label,
            "exact_match": exact_match,
            "is_correct": exact_match == 1.0,
            "latency_ms": metadata.get("latency_ms", 0),
            "input_tokens": metadata.get("input_tokens", 0),
            "output_tokens": metadata.get("output_tokens", 0),
            "raw_output": prediction.get("raw", ""),
        }

    def aggregate(self, results: list) -> dict:
        """聚合所有结果"""
        total = len(results)
        correct = sum(1 for r in results if r["is_correct"])

        # 按类别聚合
        by_category = defaultdict(lambda: {"total": 0, "correct": 0})
        for r in results:
            cat = r["category"]
            by_category[cat]["total"] += 1
            if r["is_correct"]:
                by_category[cat]["correct"] += 1

        latencies = [r["latency_ms"] for r in results]
        latencies.sort()

        return {
            "total_cases": total,
            "correct": correct,
            "accuracy": correct / total if total > 0 else 0,
            "by_category": {
                cat: {
                    "total": v["total"],
                    "correct": v["correct"],
                    "accuracy": v["correct"] / v["total"] if v["total"] > 0 else 0
                }
                for cat, v in by_category.items()
            },
            "avg_latency_ms": sum(latencies) / len(latencies) if latencies else 0,
            "p50_latency_ms": latencies[len(latencies)//2] if latencies else 0,
            "p95_latency_ms": latencies[int(len(latencies)*0.95)] if latencies else 0,
            "total_tokens": sum(r["input_tokens"] + r["output_tokens"] for r in results),
        }
```

    def __init__(self, output_dir: str = "results"):

        report = {

            "details": detailed_results,

        md_content = self._generate_markdown(summary, run_config)
        with open(md_path, "w", encoding="utf-8") as f:

from harness.input_layer import InputLayer
from harness.model_layer import ModelLayer
from harness.output_layer import OutputLayer
from harness.evaluator import Evaluator
from harness.reporter import Reporter

class HarnessRunner:
    def __init__(self, model: str = "gpt-4o-mini"):
        self.input_layer = InputLayer()
        self.model_layer = ModelLayer(model=model)
        self.output_layer = OutputLayer()
        self.evaluator = Evaluator()
        self.reporter = Reporter()

    def run(self, template_name: str, dataset_path: str,
            task_type: str = "classification") -> str:
        """运行完整评估流程"""
        # 1. 加载测试数据
        test_cases = self.input_layer.load_dataset(dataset_path)
        print(f"加载 {len(test_cases)} 条测试用例")

        # 2. 逐条执行
        results = []
        for i, case in enumerate(test_cases):
            # 2a. 渲染 prompt
            prompt = self.input_layer.render_prompt(
                template_name,
                {"email_text": case["text"]}
            )

            # 2b. 调用模型
            model_output = self.model_layer.generate(prompt)

            # 2c. 解析输出
            prediction = self.output_layer.parse(
                model_output["text"], task_type
            )

            # 2d. 评估
            result = self.evaluator.evaluate_case(
                case, prediction, model_output
            )
            results.append(result)

            status = "✓" if result["is_correct"] else "✗"
            print(f"  [{i+1}/{len(test_cases)}] {status} "
                  f"expected={result['expected']} pred={result['predicted']} "
                  f"({result['latency_ms']:.0f}ms)")

        # 3. 聚合结果
        summary = self.evaluator.aggregate(results)

        # 4. 生成报告
        report_path = self.reporter.generate(
            summary, results,
            run_config={
                "model": self.model_layer.model,
                "template": template_name,
                "dataset": dataset_path,
                "task_type": task_type,
            }
        )

        print(f"\n准确率: {summary['accuracy']:.1%} "
              f"({summary['correct']}/{summary['total_cases']})")
        print(f"报告: {report_path}")

        return report_path

**2. 并行执行设计（30 min）**

```python
import asyncio
from asyncio import Semaphore

class AsyncModelLayer:
    def __init__(self, model: str = "gpt-4o-mini",
                 max_concurrent: int = 5, max_retries: int = 3):
        self.model = model
        self.semaphore = Semaphore(max_concurrent)
        self.max_retries = max_retries

    async def generate(self, prompt: str) -> dict:
        """异步生成，带并发控制和重试"""
        async with self.semaphore:
            return await self._generate_with_retry(prompt)

    async def _generate_with_retry(self, prompt: str) -> dict:
        """指数退避重试"""
        import time
        for attempt in range(self.max_retries):
            try:
                start = time.time()
                # 使用异步 OpenAI client
                response = await self._call_api(prompt)
                latency_ms = (time.time() - start) * 1000
                return {
                    "text": response.choices[0].message.content,
                    "latency_ms": latency_ms,
                    "input_tokens": response.usage.prompt_tokens,
                    "output_tokens": response.usage.completion_tokens,
                    "retries": attempt,
                }
            except Exception as e:
                if attempt == self.max_retries - 1:
                    return {
                        "text": "",
                        "error": str(e),
                        "latency_ms": 0,
                        "retries": attempt,
                    }
                # 指数退避：1s, 2s, 4s, ...
                wait_time = 2 ** attempt
                print(f"  重试 {attempt+1}/{self.max_retries}，等待 {wait_time}s: {e}")
                await asyncio.sleep(wait_time)

    async def _call_api(self, prompt: str):
        """实际 API 调用（需安装 openai>=1.0）"""
        from openai import AsyncOpenAI
        client = AsyncOpenAI()
        return await client.chat.completions.create(
            model=self.model,
            temperature=0.0,
            messages=[{"role": "user", "content": prompt}]
        )


class AsyncHarnessRunner:
    def __init__(self, model: str = "gpt-4o-mini", max_concurrent: int = 5):
        self.input_layer = InputLayer()
        self.model_layer = AsyncModelLayer(model=model, max_concurrent=max_concurrent)
        self.output_layer = OutputLayer()
        self.evaluator = Evaluator()
        self.reporter = Reporter()

    async def run(self, template_name: str, dataset_path: str) -> str:
        test_cases = self.input_layer.load_dataset(dataset_path)
        print(f"加载 {len(test_cases)} 条测试用例，并发数: {self.model_layer.semaphore._value}")

        # 并行执行所有用例
        tasks = [
            self._run_single(template_name, case, i, len(test_cases))
            for i, case in enumerate(test_cases)
        ]
        results = await asyncio.gather(*tasks)

        # 聚合 + 报告
        summary = self.evaluator.aggregate(results)
        report_path = self.reporter.generate(summary, results, {})
        print(f"\n准确率: {summary['accuracy']:.1%}")
        return report_path

    async def _run_single(self, template_name, case, idx, total):
        prompt = self.input_layer.render_prompt(template_name, {"email_text": case["text"]})
        model_output = await self.model_layer.generate(prompt)
        prediction = self.output_layer.parse(model_output.get("text", ""))
        result = self.evaluator.evaluate_case(case, prediction, model_output)
        status = "✓" if result["is_correct"] else "✗"
        print(f"  [{idx+1}/{total}] {status} {result['case_id']}")
        return result
```

```python
import sqlite3
from datetime import datetime

class ResultStore:
    def __init__(self, db_path: str = "results/harness.db"):
        self.conn = sqlite3.connect(db_path)
        self._init_db()

    def _init_db(self):
        self.conn.executescript("""
            CREATE TABLE IF NOT EXISTS runs (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                timestamp TEXT NOT NULL,
                model TEXT NOT NULL,
                task TEXT NOT NULL,
                template TEXT,
                total_cases INTEGER,
                accuracy REAL,
                avg_latency_ms REAL,
                config_json TEXT
            );

            CREATE TABLE IF NOT EXISTS case_results (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                run_id INTEGER NOT NULL,
                case_id TEXT NOT NULL,
                category TEXT,
                expected TEXT,
                predicted TEXT,
                is_correct INTEGER,
                latency_ms REAL,
                input_tokens INTEGER,
                output_tokens INTEGER,
                raw_output TEXT,
                FOREIGN KEY (run_id) REFERENCES runs(id)
            );
        """)

    def save_run(self, summary: dict, results: list, config: dict) -> int:
        """保存一次评估运行"""
        cursor = self.conn.cursor()
        cursor.execute("""
            INSERT INTO runs (timestamp, model, task, template, total_cases, accuracy, avg_latency_ms, config_json)
            VALUES (?, ?, ?, ?, ?, ?, ?, ?)
        """, (
            datetime.now().isoformat(),
            config.get("model", ""),
            config.get("task", ""),
            config.get("template", ""),
            summary["total_cases"],
            summary["accuracy"],
            summary["avg_latency_ms"],
            json.dumps(config)
        ))
        run_id = cursor.lastrowid

        for r in results:
            cursor.execute("""
                INSERT INTO case_results (run_id, case_id, category, expected, predicted, is_correct, latency_ms, input_tokens, output_tokens, raw_output)
                VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
            """, (
                run_id, r["case_id"], r["category"], r["expected"],
                r["predicted"], int(r["is_correct"]), r["latency_ms"],
                r["input_tokens"], r["output_tokens"], r["raw_output"]
            ))
        self.conn.commit()
        return run_id

    def query_by_model(self, model: str) -> list:
        """按模型查询历史结果"""
        cursor = self.conn.cursor()
        cursor.execute("""
            SELECT timestamp, accuracy, avg_latency_ms
            FROM runs WHERE model = ?
            ORDER BY timestamp
        """, (model,))
        return cursor.fetchall()

    def find_regressions(self, run_id: int, baseline_run_id: int) -> list:
        """找出退化的用例"""
        cursor = self.conn.cursor()
        cursor.execute("""
            SELECT curr.case_id, curr.expected, curr.predicted, curr.is_correct,
                   prev.predicted, prev.is_correct
            FROM case_results curr
            JOIN case_results prev ON curr.case_id = prev.case_id
            WHERE curr.run_id = ? AND prev.run_id = ?
              AND prev.is_correct = 1 AND curr.is_correct = 0
        """, (run_id, baseline_run_id))
        return cursor.fetchall()
```



**5. 退化检测实现（15 min）**

```python
class RegressionDetector:
    def __init__(self, store: ResultStore):
        self.store = store

    def detect(self, current_run_id: int, baseline_run_id: int,
               threshold: float = 0.05) -> dict:
        """检测退化"""
        # 获取两次运行的摘要
        cursor = self.store.conn.cursor()
        cursor.execute("SELECT accuracy FROM runs WHERE id = ?", (current_run_id,))
        current_acc = cursor.fetchone()[0]
        cursor.execute("SELECT accuracy FROM runs WHERE id = ?", (baseline_run_id,))
        baseline_acc = cursor.fetchone()[0]

        # 整体退化检测
        delta = baseline_acc - current_acc
        is_degraded = delta > threshold

        # 逐用例退化检测
        regressed_cases = self.store.find_regressions(current_run_id, baseline_run_id)

        return {
            "is_degraded": is_degraded,
            "baseline_accuracy": baseline_acc,
            "current_accuracy": current_acc,
            "delta": delta,
            "threshold": threshold,
            "regressed_case_count": len(regressed_cases),
            "regressed_cases": [
                {"case_id": r[0], "expected": r[1], "was_correct": r[4],
                 "now_wrong": r[3]}
                for r in regressed_cases
            ],
        }
```

**4. 回归测试设计（15 min）**

`scripts/compare_baseline.py`：

```python
import json
import sys
import argparse
from pathlib import Path

def compare(current_path, baseline_path, threshold):
    """对比当前结果与基线"""
    with open(current_path / "results_latest.json") as f:
        current = json.load(f)
    with open(baseline_path / "results_latest.json") as f:
        baseline = json.load(f)

    current_acc = current["summary"]["accuracy"]
    baseline_acc = baseline["summary"]["accuracy"]
    delta = baseline_acc - current_acc  # 正数 = 退化

    comparison = {
        "baseline_accuracy": baseline_acc,
        "current_accuracy": current_acc,
        "delta": delta,
        "threshold": threshold,
        "is_degraded": delta > threshold,
        "regressed_cases": [],
        "improved_cases": [],
    }

    # 逐用例对比
    baseline_cases = {c["case_id"]: c for c in baseline["details"]}
    for curr_case in current["details"]:
        case_id = curr_case["case_id"]
        if case_id in baseline_cases:
            base_case = baseline_cases[case_id]
            if base_case["is_correct"] and not curr_case["is_correct"]:
                comparison["regressed_cases"].append({
                    "case_id": case_id,
                    "expected": curr_case["expected"],
                    "was_correct": base_case["predicted"],
                    "now_wrong": curr_case["predicted"],
                })
            elif not base_case["is_correct"] and curr_case["is_correct"]:
                comparison["improved_cases"].append(case_id)

    # 如果退化，创建标记文件
    if comparison["is_degraded"]:
        Path("results/regression_detected").touch()

    # 保存对比结果
    with open("results/comparison.json", "w") as f:
        json.dump(comparison, f, indent=2, ensure_ascii=False)

    # 打印摘要
    print(f"基线准确率: {baseline_acc:.1%}")
    print(f"当前准确率: {current_acc:.1%}")
    print(f"变化: {delta:+.1%} ({'退化' if delta > 0 else '提升'})")
    print(f"退化用例数: {len(comparison['regressed_cases'])}")
    print(f"改善用例数: {len(comparison['improved_cases'])}")

    if comparison["is_degraded"]:
        print(f"\n⚠️  检测到退化！准确率下降 {delta:.1%}，超过阈值 {threshold:.1%}")

    return comparison

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--current", required=True)
    parser.add_argument("--baseline", required=True)
    parser.add_argument("--threshold", type=float, default=0.05)
    parser.add_argument("--output", default="results/comparison.json")
    args = parser.parse_args()

    compare(Path(args.current), Path(args.baseline), args.threshold)
```

**6. 成本控制策略（20 min）**

```python
# 成本估算
class CostEstimator:
    PRICING = {
        "gpt-4o": {"input": 2.50, "output": 10.00},  # per 1M tokens
        "gpt-4o-mini": {"input": 0.15, "output": 0.60},
        "gpt-3.5-turbo": {"input": 0.50, "output": 1.50},
    }

    @classmethod
    def estimate(cls, model: str, num_cases: int,
                 avg_input_tokens: int = 200,
                 avg_output_tokens: int = 50) -> float:
        """估算单次运行成本"""
        pricing = cls.PRICING.get(model, cls.PRICING["gpt-4o-mini"])
        input_cost = (num_cases * avg_input_tokens / 1_000_000) * pricing["input"]
        output_cost = (num_cases * avg_output_tokens / 1_000_000) * pricing["output"]
        return input_cost + output_cost

# 成本控制策略
"""
1. CI 中只跑 smoke test（10 条）
   - gpt-4o-mini: ~$0.0004/次 → 每天约 $0.01

2. 缓存模型响应
   - 相同 prompt + 参数 → 直接返回缓存结果
   - 节省 70-90% 的 API 调用

3. smoke test 用便宜模型
   - smoke test: gpt-4o-mini
   - full test: gpt-4o

4. 定时 full test 而非每次 PR
   - PR: smoke test（10 条，~$0.001）
   - 每天 2:00: full test（35 条，~$0.003）
   - 发布前: release test（全量 + 对抗，~$0.01）
"""
```

缓存实现：

```python
import hashlib
import json
from pathlib import Path

class ResponseCache:
    def __init__(self, cache_dir: str = ".cache/responses"):
        self.cache_dir = Path(cache_dir)
        self.cache_dir.mkdir(parents=True, exist_ok=True)

    def _key(self, prompt: str, model: str, **params) -> str:
        content = json.dumps({"prompt": prompt, "model": model, **params},
                            sort_keys=True)
        return hashlib.sha256(content.encode()).hexdigest()

    def get(self, prompt: str, model: str, **params) -> dict | None:
        key = self._key(prompt, model, **params)
        path = self.cache_dir / f"{key}.json"
        if path.exists():
            return json.loads(path.read_text())
        return None

    def set(self, prompt: str, model: str, response: dict, **params):
        key = self._key(prompt, model, **params)
        path = self.cache_dir / f"{key}.json"
        path.write_text(json.dumps(response, ensure_ascii=False))
```

## 六、常见错误预警（学习过程中对照检查）


| #   | 常见错误                             | 怎么避免                                 | 在哪次容易出现 |
| --- | -------------------------------- | ------------------------------------ | ------- |
| 1   | 把 harness 当成框架来学，纠结于某个框架的 API 细节 | 先理解概念，再用框架实践；框架是实现手段                 | 第 1、7 次 |
| 2   | 只做自动评估，不做人工抽检                    | 自动评估有盲区，定期人工抽检 10-20 条               | 第 3 次之后 |
| 3   | 测试用例太少，覆盖面不够                     | 至少包含正常、边界、对抗三类，每类不少于 5 条             | 第 4 次   |
| 4   | 评估指标选错，导致 metric gaming          | 多维度评估，不要只优化一个指标                      | 第 3 次   |
| 5   | 没有基线，退化检测无从谈起                    | 第一次运行的结果存为基线，后续对比                    | 第 5 次之后 |
| 6   | CI 中跑全量测试，成本爆炸                   | 分级测试：smoke test 走 CI，full test 走定时任务 | 第 8 次   |
| 7   | 结果只看平均分，忽略分布                     | 关注 P50/P95/P99，检查是否有长尾问题             | 第 6 次之后 |
| 8   | 代码一次性写完不测试                       | 每写完一个模块就单独测试，再串联                     | 第 5、9 次 |
| 9   | prompt 模板没有版本管理                  | 从第一天就用版本号，每次改动都记录                    | 第 2 次之后 |
| 10  | 忽略数据污染风险                         | 保持测试集私密，定期检查与训练集的重叠                  | 第 4 次   |
